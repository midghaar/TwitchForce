TwitchForce

TwitchForce is my first angularjs app, and was made for educational purposes. It allows the user to view four Twitch streams simultaneously. The upper left button toggles a cinematic mode, which minimizes the ui. In the streams view, while not in cinematic mode, you can edit which Twitch channels are shown via the Edit button. Via the button below, you can also set the background image to any web url.
